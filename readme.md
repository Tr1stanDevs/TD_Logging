TD_Logging
==========

TD_Logging is a powerful logging library that helps you with logging.

Installation
------------

### What you need

- **CMake** – Version 3.15 or higher
- **Compiler** – e.g., Clang, GCC, or MSVC

## Setting things up

Clone with:
```
git@github.com:Tr1stanDevs/TD_Logging.git
```

Then go in the Project with:
```
cd TD_Logging
```

### How to Build

The build Process is fully automated.

On Windows you start with:
```powershell
    .\test-windows.ps1
```

On Windows you start with:
```shell
    .\test-linux.sh
```
